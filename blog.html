Question-1: What are the different ways to select an element in the DOM?

Within the various methods to select a DOM element, which consists of the following are: using an ID (getElementById), by class name (getElementsByClassName), by tag name (getElementsByTagName), and by name attribute (getElementsByName), CSS selectors (querySelector) for 1 and (querySelectorAll) for many. Also, elements can be selected through XPath with (document.evaluate) that makes you able to choose the items by executing the particular XPath expressions. But, the ones popular and the go-to choice are (querySelector) and (querySelectorAll) in modern web application development because of the adaptability of those selections.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

(innerHTML) can both handle the text and HTML very well as it renders any of the tags in the inside. (innerText) is the one that can only get the visible texts hiding the elements & apart from that, it also strips the HTML tags. (textContent) retrieves all the text, including hidden elements, without any HTML processing. So to say (innerHTML) is for HTML insertion, (innerText) is for only visible text, and (textContent) is for raw text, containing hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is an approach wherein an element manages events for a child element with the help of event bubbling that occurs. A single parent listener with this capability is enough to recognize the event and identify the target or child element instead of putting listeners to each child. The performance of the system is boosted and it functions even for the elements which are introduced in the dynamic way.

Question-4: What is event bubbling in the DOM?

Event bubbling is when the event gets started at the target element and comes through its parent elements. This gives the parent elements a chance to manage events that are caused by their kids. Bubbling cannot be stopped by means of parent's "event.stopPropagation()", which is fully up to the child.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create HTML components in JavaScript by using the "document.createElement()" method, you can insert them using the functions "appendChild()", "append()", or "insertBefore()", while you can delete them using the "removeChild()" function or by calling the "remove()" method on the specific element. They make it possible to do dynamic changes to the DOM.
